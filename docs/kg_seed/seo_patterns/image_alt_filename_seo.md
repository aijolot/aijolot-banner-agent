Beauty brands like Drunk Elephant, NaturaGlow, and Kylie Jenner generate significant discovery traffic via Google Image Search — shoppers searching for product images find the brand. Two factors control image indexability: the file name and the alt text. Shopify CDN serves images at URLs that include the original filename. Meaningful filenames: "naturaglow-body-wash-hero.jpg" outperforms "Screenshot 2026-05-29 at 8.21.37 p.m..png" for SEO. Alt text should describe the image content and include the product name: `alt="NaturaGlow Body Wash — nourishing soap on bathroom shelf"`. Banner agent should validate that uploaded images have descriptive alt text and flag filenames that are generic screenshot names from the Drive corpus.
