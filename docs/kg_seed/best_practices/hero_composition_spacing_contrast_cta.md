Build the hero on a clear visual flow: headline → subhead → single primary CTA, with the headline as the bold, high-contrast anchor surrounded by whitespace so it dominates. Use a consistent vertical rhythm — roughly 24–40px between headline, supporting line, and CTA — and keep margin around the whole block so it breathes. When copy sits over a photo, video, or gradient, guarantee readability with a scrim or sufficient contrast; low contrast makes even a beautiful hero unusable. Choose exactly one primary CTA and let secondary actions read as clearly lower-emphasis. On mobile, collapse to a single column and reorder so it stacks headline → image → CTA (never image-first), and scale type responsively — a 72px desktop headline becomes ~36–42px on mobile without losing its weight. The hero's type scale also sets the ceiling for the rest of the page: H2s below must read as clearly secondary to the hero headline.
