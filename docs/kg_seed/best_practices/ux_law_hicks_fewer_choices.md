Hick's Law: decision time rises with the number and complexity of choices presented — more options means more hesitation and decision paralysis. In banners this translates directly to call-to-action discipline: multiple competing CTAs dilute focus and slow (or stop) the click, so default to one primary CTA per banner and demote everything else. Where more options are unavoidable, simplify and curate the set, and use progressive disclosure — reveal advanced or secondary paths only when needed rather than crowding the primary surface. This is the conversion rationale behind one-message-one-CTA heroes and is the active lever the banner agent should apply whenever a brief tempts it to add a second or third button.
