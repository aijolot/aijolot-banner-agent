  applicable_when: "hero_main — whenever multiple messages compete for the hero slot"

Carousels split the hero's single most valuable conversion moment across slides that almost no one sees. Erik Runyon's Notre Dame data found the homepage slider drew a click-through rate of barely 1%, and 84% of those clicks landed on the very first slide — across five sites the clicks were heavily concentrated on slide one regardless of total CTR. Rotating heroes also auto-advance away from the message a user was reading, harm accessibility (motion, focus order), hurt LCP/performance by loading multiple large images, and read as advertising — compounding banner blindness. The remedy is a single static hero carrying the one most important message and CTA. If multiple campaigns genuinely need exposure, surface them as distinct sections further down the page rather than stacking them into one rotating slot where only the first is effectively visible.
