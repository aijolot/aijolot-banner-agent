In the best-performing beauty banners, the brand name (WESSAINT AURENT, Kylie Jenner Cosmetics) appears in smaller type above the headline or as a branded wordmark — never as the largest element. The H1 carries the message ("HYDRATE, TINT, GO"), while the brand name functions as provenance. This hierarchy prioritises consumer benefit over brand vanity and avoids H1 duplication with the page title, which is a common SEO and accessibility failure in Shopify section templates.
