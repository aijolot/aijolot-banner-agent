The most consistent hero layout in the beauty corpus is a 50/50 horizontal split: product image occupies one Bootstrap col-12 col-md-6 column, while headline, supporting copy, and CTA occupy the other. This pattern, seen in the BRANCY hero HTML and NaturaGlow Body Wash banner, ensures the product is always visible on desktop while copy remains legible without text overlay. On mobile (< 768px), both columns stack to full width — product image stacks above copy, or copy above image depending on whether the visual or the message is the primary hook for that campaign.
